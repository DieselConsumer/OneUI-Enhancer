extra.zip: Extra plugins (binarys)

extra.zip is needed for:

32 and 64bit devices:
   - adb commands
   
Only 64bit devices:
   - simg2img commands
   - unlock_all action
   - unlock action
   - superrepack action
   - superunpack action
   - get_offset action
   - get_size action
   - get_total_size action
   - get_all_subparts action
   - get_group action
   - checksuper action
   - start_loop action
   - get_virtual_points action
   - try_mount (NOT ALL) only with flag -file "/sdcard/super.img" (It need start_loop action to work with external super images)   
   - Sparse IMG support for update/update_zip/update_addon functions


If you are not going to use any of these actions you can delete extra.zip
